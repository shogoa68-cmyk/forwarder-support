## 単体起動

ローカル HTTP サーバ越しなら従来通り動作します（`../フォワーダー支援/shared/*.js` を
相対参照しているため、ディレクトリ構造を変えないこと）：

```sh
cd ..  # github/202605_コード改修
python3 -m http.server 18765
# → http://localhost:18765/見積支援/index.html
```
